# esports_rocketSeat
Aplicação de um site onde se encontra parceiros para jogar junto através de anuncios no site.
Projeto construído no evento next level week da Rocketseat!


## Instalação Front End
Partindo da base do projeto:
```bash
$ cd web
$ npm install
$ npm run dev
```


## Instalação Back End
Partindo da base do projeto:
```bash
$ cd server
$ npm install
$ npm run dev
```
